Актуальность:
Проблема: Существующие платформы для общения (Discord, Reddit) имеют слишком плоский, современный и унифицированный дизайн (Flat Design). Любители ламповой ретро-футуристичной эстетики и стимпанка теряют ощущение «клуба по интересам» на стандартных сайтах.
Решение: Создание тематического веб-форума с уникальным кастомизированным интерфейсом в стиле викторианского стимпанка, объединяющего атмосферу механики и современные технологии веб-разработки.



Цели и задачи:
Цель: Разработать легковесный, отзывчивый и функциональный веб-форум с глубокой визуальной стилизацией под эпоху пара и шестеренок.
Задачи:

Написать бэкенд на Django для управления ветками форума, темами и комментариями (CRUD).

Разработать аутентичный интерфейс с использованием кастомных CSS-стилей (цвета: латунь, медь, кожа, состаренная бумага).

Реализовать систему ролей пользователей (например: «Механик», «Изобретатель», «Авиатор»).

Покрыть ключевой функционал бэкенда автотестами.



Используемые технологии:
Backend: Python, Django

Database: SQLite (для разработки) / PostgreSQL

Frontend: HTML5, CSS3 (кастомные стили, анимация вращения шестеренок)



Тестирование: pytest / unittest

# Модель категории/топика форума
class Topic(models.Model):
    title = models.CharField(max_length=200)       # "Паровые двигатели и дирижабли"
    description = models.TextField()                # "Обсуждение чертежей и механики"
    created_at = models.DateTimeField(auto_now_add=True)

# Модель сообщения (поста) внутри топика
class Post(models.Model):
    topic = models.ForeignKey(Topic, on_delete=models.CASCADE, related_name='posts')
    author = models.ForeignKey(User, on_delete=models.CASCADE)
    content = models.TextField()                     # Текст сообщения
    created_at = models.DateTimeField(auto_now_add=True)

# Присвоение стимпанк-ранга пользователю на основе количества постов
def get_steampunk_rank(user_post_count):
    if user_post_count > 100:
        return "Главный Инженер Воздушного Флота"
    elif user_post_count > 50:
        return "Часовщик-Испытатель"
    return "Подмастерье у котла"
